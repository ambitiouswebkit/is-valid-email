# is Valid Email

Validation for e-mail input.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
var emailInput = getElementById(#contact-email);
isValid.email(emailInput);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
